Hello, the python scrip videochessboard_generator.py make a chessboard pattern made of number of squares for each side at your choice.
Thanks to opencv every squares is a stream of your webcam.
To close the windows, press Q on the keyboard.

Tu run the script you need to have python 3 installed on your computer, and the following python packages intalled using PIP:
- opencv-pyton
- numpy
You need a webcam on your computer.
